## Pandemic Forecasting

Covid Daily Cases Forecasting App - Back End

<!-- GETTING STARTED -->
## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone NicolasSponton/PandemicForecasting---BackEnd
   ```
2. Install packages
   ```sh
   go get
   ```
3. run app
   ```sh
   go run main.go
   ```
